ALPHA 8
=====

    Qwen3 Coder 480B Instruct (non-thinking)

  - strips out comments and renames from current function to avoid influencing LLM's re-analysis
  - you can guide LLM's re-analysis with a custom prompt + context from the current function
  - focuses only on current open function
  - "manual thinking" allows greater control over AI's thinking process
  - TESTED AND WORKS WELL WITH:
    Qwen3 235B Instruct 2507 (non-thinking)
    Qwen3 Coder 480B Instruct (non-thinking)
